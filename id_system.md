# ArCode ID System (ACS_ID)

## الشكل الرسمي:
ACS-XXX-0000

## التفسير:
- ACS = ArCode Standard
- XXX = كود المجال
- 0000 = رقم تسلسلي

## أمثلة:

- ACS-CS-0001 → Computer Science
- ACS-AI-0001 → Artificial Intelligence
- ACS-SEC-0001 → Cybersecurity
- ACS-MATH-0001 → Mathematics

## قواعد:
- لا يسمح بالتكرار
- لا يتم حذف أي ID بعد اعتماده
- أي تعديل = إصدار جديد وليس تغيير الأصل
