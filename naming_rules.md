# ArCode Naming Rules

## 1. المصطلحات
- يجب أن يكون لكل مصطلح:
  - اسم عربي واضح
  - اسم إنجليزي علمي

## 2. الكتابة العربية
- يجب استخدام العربية الفصحى فقط
- يمنع اللهجات

## 3. الكتابة الإنجليزية
- مصطلحات علمية دقيقة فقط
- بدون اختصارات غير مفهومة

## 4. المجال (DOMAIN)
- كل مصطلح ينتمي إلى مجال واحد فقط

## 5. التعاريف
- قصيرة وواضحة
- بدون غموض
- بدون فلسفة زائدة
